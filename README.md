# discord-emoji-stealer

This script allows you to download multiple emojis from many Discord servers at once easily... as long as you know how to use a computer.

## Requirements
- [Your user](https://github.com/thunder04/discord-emoji-stealer#how-can-i-get-my-user-token) or [your bot](https://discord.com/developers/applications)'s token
- [Developer Mode](https://support.discord.com/hc/en-us/articles/206346498-Where-can-I-find-my-User-Server-Message-ID-) enabled
- [Node JS](https://nodejs.org/en/download/) and [Git](https://git-scm.com/) installed
- A terminal
- An internet connection :v)

## Installation
Open your terminal and run:
- `git clone https://github.com/thunder04/discord-emoji-stealer` to clone the repository
- `cd discord-emoji-stealer` to navigate to the repository folder
- `npm install` to install all the required dependencies and build the script.

## Usage
Open your terminal and run `node .`

## FAQs

### How can I get my user token?
Using user tokens is not recommended, using them inappropriately can lead to an [account termination](https://discord.com/guidelines). If you plan to download a single server's emojis which is either public or you have the `Manage Server` permission, it is recommended to [create a bot](https://discordjs.guide/preparations/setting-up-a-bot-application.html) instead.
